# shopping-list

## Installation

- Install Dependencies : `npm i`
-
- Before packing, be sure to set `NODE_ENV` to `production` :

  **Linux :**

  ```bash
  $ export NODE_ENV=production
  ```

  **Windows :**

  ```cmd
  SET NODE_ENV=production
  ```

  or put this line in `main.js` :

  ```javascript
  process.env.NODE_ENV = "production";
  ```
